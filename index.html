Personal Online Trading Journal — ใช้งานได้จากทุกอุปกรณ์

1 สร้าง Google OAuth Client ID (ครั้งเดียว)

เข้าไปที่ Google Cloud Console แล้วทำตามนี้:

  1. สร้าง Project ใหม่ (ชื่ออะไรก็ได้ เช่น "Trading Journal")
  2. ไปที่ APIs & Services → Library → เปิดใช้ Google Sheets API และ Google Drive API
  3. ไปที่ APIs & Services → Credentials → Create Credentials → OAuth client ID
  4. Application type: Web application
  5. Authorized JavaScript origins: ใส่ URL ที่คุณเปิดไฟล์นี้
    เช่น https://yourname.github.io หรือ http://localhost:8080
  6. ไปที่ OAuth consent screen → เพิ่ม email ของคุณเป็น Test user
  7. Copy Client ID มาวางด้านล่าง
